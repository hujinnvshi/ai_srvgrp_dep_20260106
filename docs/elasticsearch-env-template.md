## 常用操作命令

### 集群健康检查
```bash
curl 'http://localhost:_______________/_cluster/health?pretty'
```

### 查看节点信息
```bash
curl 'http://localhost:_______________/_cat/nodes?v'
```

### 查看索引列表
```bash
curl 'http://localhost:_______________/_cat/indices?v'
```

### 创建索引
```bash
curl -X PUT 'http://localhost:_______________/my_index' -H 'Content-Type: application/json' -d '{
  "settings": {
    "number_of_shards": 1,
    "number_of_replicas": 0
  }
}'
```

### 插入文档
```bash
curl -X POST 'http://localhost:_______________/my_index/_doc/1' -H 'Content-Type: application/json' -d '{
  "title": "测试文档",
  "content": "这是一个测试"
}'
```

### 搜索文档
```bash
curl -X POST 'http://localhost:_______________/my_index/_search' -H 'Content-Type: application/json' -d '{
  "query": {
    "match_all": {}
  }
}'
```
